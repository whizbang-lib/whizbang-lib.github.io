Core Capabilities

Poly-Store Document + Event Store

JSONB for Postgres + native JSON columns on SQL Server/MySQL, plus a binary codec for LiteFS.

Unified Messaging Pipeline

Inline → durable → batched modes are toggleable. Built-in outbox/inbox, delayed delivery, sagas.

Handlers = Functions

Plain C# methods return results or new messages. Mark a handler as pure and the compiler forbids hidden side effects.

Cloud-Native Superpowers

Kubernetes Operator for auto-scaling projections by lag, partition-aware consumers, and seamless multi-region DR.
